# Tops Last.FM

- **Para inicializar:**

  - **NPM:**
    - `npm i`
  - **YARN:**
    - `yarn`

- **Para buildar:**

  - **NPM:**
    - `npm run build`
  - **YARN:**
    - `yarn build`

- **Para subir servidor express para servir o conteudo buildado:**

  - **NPM:**
    - `npm run server`
  - **YARN:**
    - `yarn server`

- **Entrar em src/client/services/apiService.js e colocar sua API da Last.fm**
- **Webapp rodará na porta 8080, link: <http://localhost:8080>**
